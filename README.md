# IMDb Search Comparison

## Team
- Francisco Iturriaga
- Murat Atar
- Tristan Cole

## Description
This project compares the performance of two data structures implemented from scratch: a Red-Black Tree and a Hash Table. Both structures store the same IMDb movie dataset using the Movie ID as the key. We compare insertion, deletion, and search performance.

## Features
- Load IMDb dataset
- Search movies by ID
- Search movies by rating
- Insert movie records
- Delete movie records
- Compare performance statistics
